# Symphony Orchestrator MVP

This repo contains a deployable Next.js implementation of a spec-aligned Symphony MVP:

- `WORKFLOW.md` front matter parsing + typed runtime config
- in-memory orchestrator runtime with dispatch + retry model
- tracker adapter seam (mock included)
- status and start APIs
- CI via GitHub Actions

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
